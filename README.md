# Rastreador de despesas

Projeto desenvolvido em python para controlar suas despesas.

Este projeto foi desenvolvido com foco em compreeender e treinar organização de códigos, separação de responsabilidade (Arquitetura MVC), modularização e boas práticas em python.


# Objetivo do projeto

Praticar e consolidar conhecimentos em:
- Estrutura de um projeto em python
- Arquitetura MVC
- Manipulação de arquivos JSON
- comentário de códigos
- lógica de programação
- organização de código


# Funcionalidades

Tem um menu em CLI que permite:
- Adicionar despesas
- Atualizar despesas
- Excluir despesas
- Visualizar todas as despesas
- Resumo de todas as despesas


# Observações

Projeto focado apenas para estudos, com expectativas de melhoras.
